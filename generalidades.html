Agenda

Objetivos

Objetivo general

Construir un GIS para la representación geográfica e interactiva de los centros educativos de El Salvador con la utilización del software TerraView de licencia libre. Aplicar los conocimientos adquiridos acerca de la creación, implementación y uso de los GIS en la resolución de una problemática identificada en El Salvador.

Objetivos Especificos

Investigar las capacidades, ventajas y desventajas y el proceso de instalación del software TerraView en comparación con otros softwares GIS.

Aplicar los conocimientos adquiridos sobre el software TerraView en la creación de una manual del funcionamiento y utilización del mismo.

Utilizar del software TerraView en la creación de un GIS entorno a resolver una problemática del país.

Publicar el GIS por medio del servidor de mapas GeoServer.

Programas usados

TerraView

Es un sistema de información geográfica (SIG) que proporciona una interfaz amigable y cuenta con una gran cantidad de capacidades, entre las cuales podemos mencionar

  • Permite ver y manipular datos vectoriales y rasterizados.
  • Permite la conversión y el intercambio de datos.
  • Cuenta con opciones para el análisis explicativo de datos espaciales, consultas espaciales y no espaciales.
  • Funcionalidades de procesamiento vectorial y raster.
  • Análisis espacio-temporal.
  • Creación de proyectos como un archivo pequeño con la extensión “.tview” el cual es el espacio de trabajo del usuario
  • Entre otros

Geoserver

Geoserver es un servidor web, que permite servir mapas y datos a los clientes como navegadores web y programas gis, entre sus características podemos mencionar.

  • Geoserver es un software libre
  • Permite servir datos vectoriales (WFS, shp) y datos raster (JPG, GeoTIFF).
  • Implementa el estándar Web Map Service (WMS).
  • Cuenta con una biblioteca de mapas gratuita llamada OpenLayers, la cual está integrada en GeoServer.
  • Se basa en GeoTools, un kit de herramientas Java GIS de código abierto.
  • Es de facil uso
  • Entre otras

MySQL

MySQL es un sistema de gestión de bases de datos relacionales (RDBMS) de código abierto respaldado por Oracle y basado en el lenguaje de consulta estructurado (SQL). MySQL funciona prácticamente en todas las plataformas, incluyendo Linux, UNIX y Windows. Aunque puede utilizarse en una amplia gama de aplicaciones, MySQL se asocia más a menudo con las aplicaciones web y la publicación en línea.

Las extensiones espaciales de MySQL permiten la generación, el almacenamiento y el análisis de datos geográficos:

  • MySQL tiene tipos de datos para representar valores espaciales que corresponden a las clases de OpenGIS. Estos tipos de datos permiten
  • Funciones para manipular valores espaciales. Estas funciones permiten trabajar con los formatos espaciales y realizar operaciones espaciales.
  • Creación de indices espaciales para mejorar el tiempo de ejecución de las consultas espaciales.

Tipos de datos espaciales que soporta MySQL

  • Geometry, almacena valores de geometria de cualquier tipo
  • Point
  • Linestring
  • Polygon
