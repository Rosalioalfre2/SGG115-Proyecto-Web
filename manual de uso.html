Guia de uso Terraview

Terraview es un programa para el desarrollo de GIS, el presente video tiene como objetivo el mostrar sus principales funciones y caracterisiticas.

Guia de uso Geoserver

Geoserver es un servidor de código abierto escrito en Java - permite a los usuarios compartir y editar datos geospaciales. Diseñado para la interoperabilidad, publica datos de las principales fuentes de datos espaciales usando estándares abiertos.
